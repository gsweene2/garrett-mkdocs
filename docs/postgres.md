### Get Familiar Commands

Action | `cmd`
-|-
List my user & connected db | `\conninfo`
List my roles | `\du+`
List schemas | `SELECT schema_name FROM information_schema.schemata`
List Tables | `\dt`
List Tables for a schema | `\dt <schema>.*`
List Tables for a schema | `\dt *.*`
